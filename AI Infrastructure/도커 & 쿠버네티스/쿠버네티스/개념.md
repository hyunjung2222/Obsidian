## FastAPI 도커 이미지를 Kubernetes에 배포
- Docker Hub에 이미지 업로드
    
    ```yaml
    docker login # 도커 허브 로그인
    docker build -t <도커허브_사용자명>/fastapi-app
    docker push <도커허브_사용자명>/fastapi-app
    ```
